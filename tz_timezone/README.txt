SUMMARY
=======

Adds a js file to the User Add and User Edit forms pages that remove certain optgroups from the timezone select options.


INSTALLATION
============

No special install steps are necessary to use this module, just install it normally,  see https://www.drupal.org/documentation/install/modules-themes/modules-8 for further information.


CONFIGURATION
=============

* open the file in this module named /js/tz_timezone.js to see instructions on what to change in that file to suit your needs.
